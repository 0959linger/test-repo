# 专属推理引擎

基于 v94 气场演化架构的自主推理引擎。不依赖 llama，纯物理驱动。

## 架构

```
文字输入
    ↓
┌───────────────────────────────────────┐
│  码点管线（即刻）       位图管线（后台）    │
│  ord序列 → 方向        Pillow渲染 → 气量  │
│  <1μs                  4-12ms/题         │
│  决定"往哪走"          决定"有多重"       │
└──────────┬──────────┬───────────────────┘
           │          │
           └────✦ 互补 ✦───┘
                   ↓
           v1中位角 + v3偏度
          （两条方向管线各偏半边）
                   ↓
          天道融合（rate=0.1）
     "高者抑之，下者举之，损有余补不足"
                   ↓
            v94 气场演化
          （水循环+凝结+级联+地震）
                   ↓
              卦象分布
```

## 核心原则

- **方向 + 气量分离**：码点管方向，位图管气量，不互相稀释
- **双管方向**：v1中位角(偏乾)+v3偏度(偏坤)，天然互补
- **天道融合** ⭐：avg打底 + 对位卦间微量流通(10%)，8/8卦全触发
- **异步解耦**：两条管线独立进化，互不影响
- **纯物理**：不查数据库、不训练、不需要 llama
- **即刻响应**：方向<1μs，位图4-12ms后台，50题<1秒
