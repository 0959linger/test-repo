# v94 全局架构 — cos/sin/weight 重新放置

## 当前架构（直接分发版）

```
7D特征
   ↓
直接分发(无cos/sin/weight)  ← 纯坐标×特征，无人为权重
   ↓
趋势旋转(几何旋转)
   ↓
【演化循环×50】
   ├─ 冲刷(扩散+聚集+衰减)    ← 纯物理力
   ├─ 级联追逐(按值排，高→低)  ← 无方向性
   ├─ 冷却度(源卦累积)        ← 对所有人都一样
   ├─ 地震(削平10%)          ← 恒定强度
   └─ 蒸汽流动(梯度)          ← 纯物理力
   ↓
太极衰减 → 温度计 → 凝结
```

## cos/sin/weight 的三个位置候选

### 位置A：级联追逐 ← cos（相错）

当前：`src→dst flow = diff × 0.4`（只看差值，不管谁是谁）

改为：`flow = diff × (1 - cos(angle_diff)) × 0.4`
- 乾⇄坤（180°对立）：cos=-1 → 1-cos=2 → 流动加倍
- 乾⇄兑（45°相邻）：cos=0.707 → 1-cos=0.293 → 流动减小
- **物理直觉：对立卦位之间流动更强，相邻卦位之间流动更弱**

### 位置B：地震 ← sin（物极必反）

当前：`地震 = 削平10%`（恒定强度）

改为：`地震强度 = sin(extremeness × π) × 削平10%`
- 极端度=1.0：sin(π)=0 → 不地震（正常）
- 极端度=1.5：sin(1.5π)=-1 → 地震最强（极阳转阴）
- 极端度=2.0：sin(2π)=0 → 地震消失（已经反转完了）
- **物理直觉：极点到顶自动衰弱，过顶地震停止**

### 位置C：冷却度 ← weight（爻位）

当前：`冷却累积 = overflow × 0.02`（所有卦位一样）

改为：`冷却累积 = overflow × 0.02 × (1 - 阳爻比例×0.5)`
- 乾（3阳）：weight=1-1.5×0.5=0.25 → 冷却很慢
- 坤（3阴）：weight=1-0×0.5=1.0 → 冷却正常
- 震（1阳2阴）：weight=1-0.5×0.5=0.75
- **物理直觉：阳爻越多冷却越慢（阳气保温），阴爻越多冷却越快**
