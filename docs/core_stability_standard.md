# v94 核心稳定性验证标准

## 背景

v94 是纯确定性偶极子场——所有机制（扩散、聚集、级联、冷却、地震、蒸汽流、太极衰减）
均为确定性函数，内部零随机源。因此：

> **"同输入重复测试"在确定性系统中是平凡的，不构成稳定性验证。**

## 真正的核心稳定性标准（四维）

### 1. 结构稳定性 (Structural Stability)
**定义**：初始 qi 加 ±ε 微扰动后，收敛结果是否保持？
**判据**：主导卦保持率 ≥ 60%（即噪声下不频繁切换卦位）
**意义**：如果大多数输入都在临界点附近（噪声导致漂移），说明场的分辨能力不够，
       需要重新审视桥接层（河图→蒸汽→qi）的精度。

### 2. 收敛稳定性 (Convergence Stability)
**定义**：对所有合法输入（含边界），v94 是否总能收敛到有意义的分布？
**判据**：零崩溃、零NaN、winner 始终是合法卦位、distribution 和为1
**意义**：这是架构底线——不能因为边界输入而崩溃。

### 3. 物理一致性 (Physical Consistency)  
**定义**：语义相近的输入 → 卦位邻近（或用相同卦位回应）？
**判据**：用八卦盘的角距离度量。同类问题组内角距离 < 异类组间角距离。
**意义**：如果同类问题的卦位散落在八卦盘各处，场不是在"理解"而是在"掷骰子"。

### 4. 单调性 (Monotonicity)
**定义**：更强的初始主导（更高的 spread）应该导致更高的收敛确信度？
**判据**：spread 与 CV 单调正相关（Spearman ρ > 0.5）
**意义**：河图输出的"确定性"应该传递到 v94 的"确定性"，
       如果不是，那中间有信息丢失。

## 测试范围

- 8 题标准集（升职/被冤枉/偷窃/美/爱/数学/说谎者/进化论）
- 每组测结构稳定性（±2%噪声×30次）
- 同类问题聚类测物理一致性
- 所有测试≥spread梯度测单调性
- 边界输入测收敛稳定性
