# 龙战 — 高共识低CV动力学态

## 定义

龙战是河图-蒸汽-v94链路中出现的一种特殊终态：

> **qi 有结构（极端度 > 3.5），但级联阈值跨不过去（级联 = 0），CV 被压死在 2.5 以下。**

物理类比：满油箱没火花塞。燃料（部落共识）极其充足，但 qi 分布不够陡峭，cascade_chase 的阈值 `diff > 0.02 × qi[强卦]` 点不着火。能量憋在 qi 里散不掉——极端度读数虚高，CV 读数趴窝。

命名来源：周易「龙战于野，其血玄黄」——群龙无首，力均而散。与"热寂"（力均而寂）对称。

## 四态对照表

|  | 热寂 | 龙战 | 平稳 | 高压 |
|---|------|------|------|------|
| **qi 结构** | 无（各卦均匀） | 有（温差存在） | 有 | 有 |
| **级联追逐** | 0 | **0** | 少许 | >3 |
| **极端度** | < 2.0 | **> 3.5** | 2~3.5 | 不定 |
| **CV** | < 2.0 | **< 2.5** | 2.5~3.5 | > 3.5 |
| **部落集中度** | < 8 | **≥ 11** | 8~12 | 不定 |
| **冷却堆积** | 无 | 稀薄（<2 卦 >0.05） | 稀疏 | 不定 |
| **物理图景** | 枯井·没水 | 满油箱·没火花 | 小火慢炖 | 大火煮沸 |

**热寂 vs 龙战的区别：**
- 热寂：6 个模型产出相似内容 → 蒸汽无温差 → qi 均匀 → 没能量可耗散
- 龙战：6 个模型产出不同内容但聚类高度集中 → 蒸汽有温差 → qi 有结构 → 级联阈值刚好跨不过去 → 能量憋着

**区分口诀：** 热寂看部落（<8），龙战看级联（=0 且部落 ≥11）。

## 触发条件

必要条件（同时满足）：

```
部落 max ≥ 11    AND    级联总数 = 0    AND    极端度 > 3.5    AND    CV < 2.5
```

其中最关键的是「级联 = 0 但极端度 > 3.5」——这意味着 qi 有足够温差让极端度显示结构，但温差不足 2% 相对差来触发 cascade_chase。

### 常见触发场景

1. **高共识问题（天然龙战体质）**
   - 1+1=2：部落 14-19，模型高度一致，但模型输出的信号相似度高，蒸汽 qi 分布不够陡
   - 道德判断题（偷窃是错误的吗）：部落 9-11，偶尔触发

2. **跨问题记忆错位（助燃剂，非根源）**
   - 前一题的强记忆偏置指向卦 X，新题的河图信号指向卦 Y
   - 记忆偏置让 qi 有了结构，但偏置方向（X）和河图方向（Y）互相拉扯 → qi 有结构但不陡
   - 这种情况下出现龙战意味着：**记忆在跨问题污染当前判断**
   - 注意：如果一个问题的河图信号方向刚好和记忆偏置一致，龙战会被"压平"为高压——同向叠加让 qi 更陡，级联能点着

3. **记忆≠龙战根源**
   - 纯 1+1=2 在零记忆下也龙战（2026-07-04 实验 2 证实）
   - 龙战的根源是河图信号的特征结构，不是记忆

## 测试中遇到龙战怎么办

### 排查步骤

1. **确认是否真龙战**：检查四条件（部落≥11、级联=0、极端>3.5、CV<2.5）
2. **检查记忆池**：当前记忆是否有跨问题偏置？
3. **检查河图原始部落分布**：K-means 聚类后各部落的信号数量
4. **检查蒸汽 qi**：8 卦位的初始 qi 值，看温差和级联阈值的距离

### 判定

| 情况 | 含义 |
|------|------|
| 纯问题自带龙战（如 1+1=2） | 正常——该问题的河图信号天然如此，不是 bug |
| 跨问题记忆错位导致龙战 | 「认知污染」——上一题情绪残留在干扰当前判断 |
| 同问题连续多轮龙战 | 异常——同问题不应该每次都龙战，检查记忆偏置是否过强 |
| 所有问题都龙战 | 严重异常——检查 qi 生成逻辑或 cascade_chase 阈值 |
